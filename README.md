# NAAL(Neuromorphic Architecture Abstraction Layer)
- Neuromorphic Architecture Abstraction Layer
- Especially for Contstrained Hardware such as IoT device

## Hardware To be Supported
- ETRI Board
- ABR FPGA Board
- Intel Loihi
